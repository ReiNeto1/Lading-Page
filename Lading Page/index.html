RPG BOT

Um Bot Voltado Para jogadores de RPG para facilitar sua historia

O que ela faz por você?

1. Cria um rolagem de dados para para a visualizaçao dos outros jogadores

2. Possui um sistema de banimento para ações suspeitas é palavras inadequaas

3. Facilita a chama do Mestre , para perguntar ou falar algo fora das partifas
